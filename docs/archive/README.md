# Archived documents

These docs reflect ThruSpark's original scope as a cross-platform product targeting Google Play and the App Store. The project has been paused and open-sourced as a portfolio piece; nothing here is being actively pursued.

Preserved as a record of the privacy/compliance thinking and iOS scoping that went into the design.

- `Compliance-Handoff.md` — pre-submission compliance reference originally written for Play Console + App Store Connect. Email addresses and privacy URLs in this doc reference a personal domain (`thebikemechanic.ca`); references to Supabase auth and account features are stale, since that integration was removed when the app was open-sourced.
- `ios/` — iOS scope notes from before any Swift code was written. The iOS app was never built; this folder is the planning artifact, not source code.
